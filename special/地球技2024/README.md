# 利用について

地球技大会PJ用ロゴ（以下：PJロゴ）の使用に関する規定をここに記す。
利用は、利用可能者が目的の範囲内でのみ可能である。それ以外は不正利用とする。

## 権利帰属
PJロゴに関する一切の権利は「GSC学生連合」（以下：学連）に帰属する。
- 著作権者：
	- **GSC学生連合**
- 著作者：
	- **地球技大会PJ**

## 利用可能者
PJロゴの利用は**学連に所属する者**に限って認められる。特例として左記の者が許可した場合、学連に所属していない者の利用を認める場合がある。

## 目的
利用可能者はその使用目的を**責任者**（地球技大会PJリーダー、執行部、顧問、またはそれに準ずる者）のいずれかに申し出た上で許可された範囲内でのみ利用できる。
#### 例：
- 地球技大会PJによる企画用ロゴを目的とした各種媒体（印刷物、広報用素材等）での利用
- 学連執行部による広報用素材を目的とした利用

## 不正利用
上記における利用可能者と目的以外の利用は不正利用とみなす。この場合、直ちに利用を停止するべし。依然として利用する場合、学連より利用停止を申し出る。
#### 例：
- 広報部による利用において申し出の無い場合
- 学連以外における利用

## 免責
学連は、PJロゴに関する瑕疵（安全性、信頼性、正確性、完全性、有効性、特定の目的への適合性、セキュリティなどに関する欠陥、エラーやバグ、権利侵害などを含む）が無いことを保証しない。学連は、上記における瑕疵を除去してPJロゴを提供する義務は無く、PJロゴに起因して利用者に生じた損害について一切の責任を負わない。

# ファイルについて
- PNG
- SVG

## 各種仕様
- PNG形式 1740px*1740px
- 基調色 #479e88（地球社会共生学部と同様）

## 色毎のファイルについて
- 2024年度第三回地球技大会PJ企画内での利用を目的としたもの。
- 「地球技大会第三回用ロゴ+<u>色（赤、緑、青、黄）</u>.拡張子」は組（団）の色に沿って作成。
- 「地球技大会第三回用ロゴ+色（赤、緑、青、黄）+<u>２</u>.拡張子」とついているものは利用にあたって視覚しやすくしたもの。

## ファイルの編集
- 利用可能者は**元データの複製**を自由に編集することが出来るが、その利用においては責任者の許可が必要である。
- 参考用作成時編集環境：「Inkscape v1.3.2」「Windows 11 23H2 x64」
- PJロゴの新規作成（二次的著作物の範囲でない創作、あるいは一新）をする場合、同フォルダ内の本ファイル（README.md）を除いた全てのファイルをアーカイブとして同フォルダ内に新規フォルダを作成し、その内に移動するべし。なおフォルダ名は「v1_archive」とする。
